# ✈ TravelFlow

TravelFlow adalah aplikasi web pemesanan wisata berbasis PHP dan MySQL yang dirancang untuk memudahkan pengguna mencari destinasi wisata, melakukan booking, melihat riwayat transaksi, serta memberikan ulasan terhadap tempat wisata yang telah dikunjungi.

## 🚀 Fitur Utama

* Login dan Registrasi Pengguna
* Dashboard Pengguna
* Daftar Destinasi Wisata
* Filter Kategori Wisata
* Booking Wisata
* Riwayat Transaksi
* Ulasan Wisata
* Profil Pengguna
* Manajemen Data Hotel
* Manajemen Kendaraan
* Notifikasi Pengguna

## 🛠️ Teknologi yang Digunakan

* PHP Native
* MySQL / MariaDB
* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* XAMPP

## 📂 Struktur Database

Database utama:

```sql
travelflow
```

Tabel yang digunakan:

* users
* wisata
* hotel
* kendaraan
* transaksi
* ulasan
* notifikasi
* paket

## ⚙️ Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/username/travelflow.git
```

2. Pindahkan folder ke:

```text
xampp/htdocs/
```

3. Buat database:

```sql
travelflow
```

4. Import file database:

```text
travelflow.sql
```

5. Sesuaikan konfigurasi database pada:

```php
config/koneksi.php
```

6. Jalankan XAMPP (Apache dan MySQL)

7. Buka browser:

```text
http://localhost/travelflow
```
